# Raspberry Pi 32-bit Assembly ARM Programming

Assembly ARM exercises for a 32-bit OS.

# How to run
Change **name** with the **name of the file you are running**.
## AS compiler
`as -o name.o name.s` <br>
`ld -o name name.o` <br>
`./name` <br>

## C compiler
`as -o name.o name.s`  <br>
`gcc -o name name.o`  <br>
`./name`
